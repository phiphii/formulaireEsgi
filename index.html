Présentation du projet .NET

1 - Introduction :

Le but de ce projet est de créé une solution de création de formulaire en ligne. Ce système proposera au minimum les fonctionnalités suivantes :
- La gestion des formulaires
- La visualisation des statistiques
- La soumission de réponse à un formulaire

2 - Livrables :

Les livrables suivant sont attendus à la fin du projet :
- Projet Visual Studio

3 - Backoffice :

3.1 – Création d’un formulaire

La création d’un formulaire permettre de renseigner les informations suivantes :
- Titre
- Description
- Date de clôture
- Liste de questions
Remarques : Pour chaque formulaire, un hash unique sera généré. Il permettra aux utilisateurs d’accéder au formulaire pour y répondre.

3.2 – Actions sur un formulaire

Pour chaque formulaire, vous devez pouvoir :
- Le clôturer
- Le modifier
- Le supprimer
- Consulter les statistiques

4 - Front :

Le front permettra à un utilisateur en possession d’une url d’accéder à un formulaire et d’y répondre. Si l’url ne correspond à aucun formulaire, alors une page d’erreur est affichée.
